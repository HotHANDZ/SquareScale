SquareScale backend is running

http://localhost:8080 is only the API (JSON). The login screen is a separate file in your project — you open it like any web page on your PC.

How to get to the login screen (do this after the backend is running)
  1. In File Explorer, go to your project folder, then open the frontend folder.
  2. Double‑click index.html (or right‑click → Open with → Chrome / Edge / Firefox).
  3. You should see the SquareScale login page. Sign in with a user from your database.

Tip (Cursor / VS Code): Right‑click frontend/index.html in the sidebar → Reveal in File Explorer, then double‑click the file.

Alternative: Install the “Live Server” extension and use “Open with Live Server” on index.html — the app still talks to http://localhost:8080.

The JavaScript in frontend/src/global.js calls http://localhost:8080/auth/login, so keep mvnw spring-boot:run running while you test.
